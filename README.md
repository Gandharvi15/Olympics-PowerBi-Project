📊 Olympics Data Analysis Dashboard – Power BI
📌 Project Overview

This project presents an interactive Olympics Data Analysis Dashboard built using Power BI.

The objective of this project is to analyze historical Olympic data to identify trends, patterns, and performance insights across countries, athletes, and sports.

The dashboard enables users to explore medal distribution, country dominance, gender participation trends, and sport-wise performance through interactive visuals and filters.

🎯 Business Objective

The primary goals of this analysis are:

Analyze medal distribution by country

Identify top-performing athletes and nations

Understand gender participation trends over time

Compare sport-wise medal performance

Track historical growth of Olympic participation

📂 Dataset Information

Dataset: Historical Olympics Dataset

Source: Kaggle

File Type: CSV

Data Includes:

Athlete Name

Age

Gender

Country

Sport

Event

Year

Medal Type

🧹 Data Cleaning & Transformation

Data cleaning and transformation were performed using Power Query in Power BI.

Steps Performed:

Removed null and missing values

Standardized country names

Converted data types (text, numeric, date)

Removed duplicate records

Created calculated columns

Created DAX measures for:

Total Medals

Gold Medals

Silver Medals

Bronze Medals

Medal Contribution %

📊 Dashboard Features

The dashboard includes:

🏅 Total Medal Summary (KPI Cards)

🌍 Country-wise Medal Distribution

🥇 Top Performing Countries

🧍 Gender Participation Analysis

🏆 Sport-wise Medal Analysis

📅 Year-wise Olympic Trends

🎛 Interactive Filters (Country, Year, Sport, Gender)

📈 Key Insights Generated

Certain countries consistently dominate medal counts across multiple years.

Male participation historically exceeded female participation, but the gap has reduced significantly in recent Olympics.

Specific sports contribute disproportionately to overall medal counts.

Medal performance trends vary significantly across decades.
